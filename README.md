# stripe.dev

https://stripe.dev landing page.

## Development

### Installation

```bash
$ git clone https://github.com/stripe/stripe.github.io.git
$ cd stripe.github.io
$ yarn
```

### Run this app

```bash
$ yarn dev
```

### Build

```bash
$ yarn build
```

You can check the static site by the following command:

```bash
$ npx serve out
```

### Lint

```bash
$ yarn lint
```
